CSS

CSS는 HTML과 완전히 다른 언어로 HTML은 정보를 담고, CSS는 정보를 꾸미는 역할을 한다. CSS 등장 이전에는 font와 같은 태그 들이 있었으나 이러한 태그들은 정보를 담고 있지 않아 정보를 담고 있는 태그와 분리할 필요성이 대두되었다. 그래서 등장한게 CSS이다. CSS는 head 태그 안에서 특정 태그에 대한 속성을 부여할 수도 있고, 기존의 태그 안에서 작동할 수도 있다. 이렇게 다른 언어를 HTML에서 읽어드릴 수 있게 하는 속성이 style 이다. 이 속성은 CSS 언어를 담고 있으므로 CSS 언어로 해석하라는 의미를 담고 있다. 이를 통해 웹은 CSS를 HTML과 같이 인식하여 읽어드릴 수 있는 것이다.

속성-style

태그의 속성 style는 그 값으로 css 효과가 반드시 들어온다는 것을 의미하고, style 태그는 선택자(태그)에게 지정되는 효과를 담는다. 속성 style는 style="효과"로 쓰여지고, 태그 style는 태그 선택자(표현되는 대상 태그) {효과;} 태그로 쓰여진다.
즉, selector{declaration}이 기본적인 형식이고, declaration은 property:value;로 구성되어 진다.

selector(선택자)과 property(효과)

이제 원하는 표현을 위해 선택자와 효과를 검색하면 된다. 예를 들어 css text size property가 있다.
HTML 속성 class를 이용하면 지정된 class를 선택하여 효과를 줄수 있다. 이를 통해 동일한 태그에 대해서 다양한 효과를 줄 수 있게 된다. 그러나 이렇게 사용되면 가장 나중에 쓰여진 효과가 나타나게 된다. 그래서 이를 해결하는 것이 id 속성이다. 즉, id 선택자가 class 선택자 태그 선택자 순으로 중요도가 높아진다.

박스모델

h1과 a 태그를 비교해 보면 h1은 화면 전체를 이용하고 a는 태그로 감싸진 크기만큼을 갖는다. 이처럼 html의 태그는 성격에 따라 화면 전체를 이용하기도 그렇지 않기도 한다. 이때 화면 전체를 사용하는 태그는 block, level, element 자기 자신의 부피만큼을 갖는 것을 inline element라 한다.

그리드

디자인을 하기 위해서는 태그가 있어야 한다. 그런데 디자인만을 위해 태그를 사용하는 것은 CSS가 만들어진 목적에 맞지 않다. 그래서 이럴때 사용할 수 있도록 있는 무색무취의 태그가 바로 div와 span이다. div는 element이고, span은 inline이다. 최신의 기술을 사용할 때는 현재사용해도 되는지 데이터를 통해 확인해야 하는데 이때 사용하는 사이트가 caniuse.com이다.
