javascript

HTML은 정보, CSS는 디자인(효과), JavaScript는 사용자와의 상호작용하는 기능을 담는다.

JavaScript 실행하기 - script 태그

그렇기에 javascript는 HTML 위에서 동작하게 되다. 이를 위해서는 CSS와 마찬가지고 자바스크립트가 시작된다는 것을 알리는 태그가 필요하다.

이벤트(event) - input 태그

웹에서 발생하는 사건들(변화?)를 이벤트라고 한다.

콘솔(console)

웹브라우저에서 검사 console을 통해서 javascript를 실행할 수 있다. 이는 이미 만들어진 웹페이지를 콘솔을 통해 간단하게 해결할 수 있게 된다.

data type - number & string

variable, constant

웹 브라우저를 동적으로 바꿔보자

조건문

black 버튼과 white 버튼이 두개 있는게 아니라 하나의 버튼으로 만들고 싶어졌다. 이런 것을 toggle이라 한다.
즉 어두운 화면에서는 white 버튼이 활성되고, 밝은 화면에서는 black 버튼이 활성되게 하는 것이다.
이를 위해서는 비교 연산자와 Boolean 데이터 타입을 알아야 한다.

refactoring

먼저 버튼이 두개가 있어 서로 id 값에서 중첩된다. 이를 해결하고 간단하게 만드는 것이 해당 태그 내에서만 작동할 수 있도록 하는 것이다. 이때 사용하는 것이 this이다.
그런데 아직도 중복되는 것이 있다. 이를 해결하는 것이 var로 정의하는 것이다.

함수(function)

특정 코드가 반복되지만 연속적으로 되어 있지 않다면 반복문을 사용하기가 힘들다. 이때 함수가 사용된다.

객체(object)

refactoring을 하기위해 function을 많이 사용하다보면 function이 서로 겹치게 될 수도 있게 된다. 이렇게 되면 함수가 충돌하게 되고, 이러면 에러가 난다.
객체는 서로 연관된 함수와 변수를 수납하는 용도로 사용되기도 한다. 폴더의 관점으로 봐도 무방하다?
객체에 포함되어 있는 함수는 method라 한다.
배열과 다르게 객체는 순서 없이 정리된다.
